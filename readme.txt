Author: Akat Merey

Descripton: 
    This web app lets users select cocktail ingredients, then fetches matching recipes from an external API.  The app uses Express.js for the backend and EJS for rendering the frontend.  Ingredients are sent to the API with axios, and the app displays the recipes or handles errors accordingly.

Used methods/tools/technics:
    - Express.js
    - EJS
    - axios
    - body-parser
    - GET request to external API

How to launch:
    1) Download all the files 
    2) In terminal write "npm i"
    3) Then in terminal "nodemon cocktail.js"
    4) Go to the browser for "http://localhost:2000/" adress
